Este documento presenta el diseño y planificación de un sistema de gestión e commerce orientado a la prestación de servicios profesionales de telecomunicaciones, cableado estructurado, seguridad electrónica y ciberseguridad. El trabajo se desarrolla bajo un enfoque de ingeniería, alineado a estándares académicos (IEEE/ISO), incorporando principios de arquitectura de software, programación funcional en Go y modelos formales de gestión de servicios.
